# American Brachytherapy Society (ABS) Consensus Statements for Prostate Brachytherapy

## Document Type
ABS official consensus statements covering LDR, HDR, and pulsed-dose-rate (PDR) prostate brachytherapy.

## LDR Monotherapy (2012 — Davis et al.)

Davis BJ, Horwitz EM, Lee WR, Crook JM, Stock RG, Merrick GS, Butler WM, Grimm PD, stone NN, Potters L, Zietman AL, Zelefsky MJ; **American Brachytherapy Society. American Brachytherapy Society consensus guidelines for transrectal ultrasound-guided permanent prostate brachytherapy.** *Brachytherapy* 2012;11(1):6-19. PMID: 22221608. DOI: 10.1016/j.brachy.2011.07.005.

### Prescription Doses (LDR monotherapy)
| Isotope | Dose |
|---------|------|
| I-125 (Advantage) | 144 Gy |
| I-125 (OncoSeed / TheraSeed) | 145 Gy |
| Pd-103 (TheraSeed) | 125 Gy |
| Cs-131 | 115 Gy |

### Dosimetric Goals
| Metric | Target |
|--------|--------|
| D90 | ≥ 100% PD (for I-125 → ≥ 144 Gy) |
| V100 | ≥ 95% |
| V150 | < 50% |
| V200 | < 20% |

## HDR Monotherapy (2020 — Yamada et al.)

Yamada Y, Rogers L, Demanes DJ, Shaikh T, Chu F, Mantz C, Danforth D, Ghanouni P, D'Amico AV, Steinberg ML, Kamrava M, Wang J, McLaughlin W, Braccioforte M, Crawford ED, Wilson T, Wachsman A, Kased N, Moghanaki D, Rettig M, McClure S, et al. **American Brachytherapy Society consensus guidelines for high-dose-rate prostate brachytherapy.** *Brachytherapy* 2020;19(5):595-613. PMID: 32694097.

### Prescription Doses (HDR monotherapy)
| Regimen | Dose | Fractions |
|---------|------|-----------|
| Single fx (palliative) | 19 Gy | 1 |
| Single fx (boost) | 15 Gy | 1 (after 37.5 Gy EBRT) |
| Ultra-fractionated | 27 Gy | 2 × 13.5 Gy |
| Moderate | 34–36 Gy | 4 × 8.5–9 Gy |
| Standard | 38 Gy | 6 × 6.35 Gy |
| Equivalent | 31.5 Gy | 7 × 4.5 Gy |

## OAR Constraints (HDR)

| Organ | Goal |
|-------|------|
| Urethra D10 | ≤ 118% Rx |
| Urethra D30 | ≤ 110% Rx |
| Rectum D2cc | ≤ 75% Rx |
| Bladder D2cc | ≤ 80% Rx |
