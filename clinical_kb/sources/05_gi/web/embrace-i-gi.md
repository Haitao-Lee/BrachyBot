# AAPM TG-229 High-Energy Brachytherapy Source Dosimetry

- **URL**: https://www.aapm.org/Reports/
- **Title**: AAPM Task Group Report 229 (TG-229): Dosimetry of High-Energy Brachytherapy Sources
- **Journal**: Medical Physics
- **Year**: 2012
- **Source type**: Physics task group report

## Scope
- Co-60 HDR (TheraGenesis / BEBIG MultiSource / Elekta Flexisource Co-60)
- Applicator-specific dosimetry
- New dose calculation formalisms
- Includes GI brachytherapy source configurations

## Key Recommendations
- Adopt TG-43 + advanced model-based dose calculation (Acuros BV / ACE)
- Acuros BV/BrachyVision ACE grid-based Boltzmann solver
- Account for tissue heterogeneity (air, bone)
- Medium-dependent correction factors
- Specific source datasets for GI intraluminal applicators

## Relevance to GI Brachytherapy
- Esophageal intraluminal applicator: validated source data
- Bile duct intraluminal: single-channel and multi-channel
- Rectal intracavitary: cylinder/endorectal applicators
- Need for explicit heterogeneity corrections near air gaps
- Applicator reconstruction uncertainty budget

## Key Findings
- Co-60 HDR provides comparable plan quality to Ir-192 with longer source life
- TG-43 overestimates dose by 1-3% in heterogeneous tissues
- ACE/Acuros algorithms preferred for accurate dose near air/tissue interfaces (common in GI)
- Independent calculation verification required
