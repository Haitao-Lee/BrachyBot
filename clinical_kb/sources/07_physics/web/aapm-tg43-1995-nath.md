## 1D Formalism

For an idealized point source, the dose rate at distance r in water is:

Ḋ(r,θ) = Λ · S_k · (G(r,θ)/G(1,θ₀)) · g(r) · F(r,θ)

Where:
- Λ = dose-rate constant (cGy·h⁻¹·U⁻¹)
- S_k = air-kerma strength (U = 1 cGy·cm²·h⁻¹)
- G(r,θ) = geometry function
- g(r) = radial dose function
- F(r,θ) = anisotropy function
- θ₀ = 90° (reference polar angle)

## Key Recommendations

- All clinical dose calculations should be based on the TG-43 formalism
- Source strength should be specified as air-kerma strength S_k rather than apparent activity
- Reference point for Λ is r=1 cm, θ=90° in water
- Traceability to a primary standards laboratory (NIST, NRC, NPL, PTB) required
