# Brachytherapy for Salivary Gland, Sinus, and Skull Base Tumors

## Salivary Gland Tumors

### Indications for BT
- Post-operative adjuvant BT for close/positive margins
- Recurrent pleomorphic adenoma
- Recurrent or residual malignancy not amenable to surgery
- Selected primary tumors (high surgical risk patients)

### Technique
- Interstitial catheters placed in tumor bed at surgery
- Or post-op image-guided placement
- Afterloading technique (HDR or PDR)

### Dose / Fractionation
- Adjuvant: 45–60 Gy EQD2 (combining EBRT + BT) or 50–60 Gy HDR alone
- Boost after EBRT: 15–25 Gy in 3–6 fx
- Pleomorphic adenoma: 50–55 Gy / 15–20 fx HDR

## Paranasal Sinus / Nasal Cavity

### Indications
- T1–T2 SCC of nasal vestibule
- T1–T2 esthesioneuroblastoma (Kadish A–B)
- Recurrent inverting papilloma
- Adjuvant after endoscopic resection (close margins)

### Technique
- Surface mold / intracavitary
- Interstitial for thicker lesions
- Custom 3D-printed applicators for nasal cavity

### Dose / Fractionation
- 50–60 Gy in 10–15 fractions HDR (monotherapy for small lesions)
- Boost after EBRT: 10–20 Gy in 2–4 fx
- Total EQD2: 70–80 Gy

## Skull Base / Temporal Bone

### Indications (highly selected)
- Recurrent chordoma / chondrosarcoma (often combined with EBRT/proton)
- Recurrent nasopharyngeal carcinoma (salvage)
- Selected pituitary / parasellar recurrences
- Selected parotid bed recurrences near skull base

### Technique
- Interstitial catheters placed at surgery
- Image-guided placement with CT/MRI fusion
- Intraoperative BT for unresectable residual

### Dose / Fractionation
- 15–25 Gy single-fraction intra-op
- 30–40 Gy in 5–10 fx HDR (post-op)
- Combined with EBRT for cumulative dose of 70–80 Gy

## OAR Constraints (Sinus / Skull Base)
| Structure | Constraint (EQD2, α/β 3) |
|---|---|
| Optic nerve / chiasm | D0.1cc ≤ 50 Gy |
| Retina | D0.1cc ≤ 45 Gy |
| Lens | D0.1cc ≤ 10 Gy |
| Lacrimal duct | D0.1cc ≤ 50 Gy |
| Brainstem | D0.1cc ≤ 54 Gy |
| Spinal cord | D0.1cc ≤ 50 Gy |
| Temporal lobe | D0.1cc ≤ 65 Gy |
| Carotid | D0.1cc ≤ 75–80 Gy cumulative |
| Cochlea | Mean < 45 Gy |
| Pituitary | Mean < 30 Gy (function) |
| Brain (whole) | V20 ≤ 30% |
