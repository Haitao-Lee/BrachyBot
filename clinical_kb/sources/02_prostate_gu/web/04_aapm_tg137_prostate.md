## Key Recommendations

### Dose Prescription Standardization
- Report mPD (minimum peripheral dose) — historically "prescription dose"
- mPD corresponds to the isodose line covering the prostate volume with margin
- For monotherapy LDR, mPD = prescription dose (e.g., 144 Gy I-125)
- Recommend: D90 ≈ mPD (homogeneous implant)

### Calculation Algorithm
- TG-43 formalism with NIST 1999 (or 2017 updated) calibration
- Use full scatter water kernel
- Report anisotropy function, radial dose function
- Account for inter-seed effects (rapid strand dosimetry)

### Volume Definition
- CTV = prostate (gross volume = CTV for LDR)
- No PTV margin (LDR is real-time with prostate motion covered)
- Prostate volume from TRUS (intra-op) or CT (post-implant)
- Use contoured volume, not ellipsoid estimation

### Reporting Parameters
Required for each implant:
- Total source strength (U or air-kerma strength, µGy⋅m²⋅h⁻¹)
- Number of sources
- Source model and lot
- D90, V100, V150, V200
- D2cc, D0.1cc for rectum, bladder, urethra
- Coverage index (CI), homogeneity index (HI)
- External volume index (EI)
- DHI (dose homogeneity index)

### Quality Metrics
- D90 ≥ 100% (target 100-110%)
- V100 ≥ 95%
- V150 ≤ 50-60% (acceptable range)
- V200 ≤ 20%
- HI = (V100 - V150)/V100
- CI = (V100/TV) × (V100/TV100ref) — not commonly used

## Impact on Clinical Practice
- Standardized dosimetry reporting
- Allows cross-institution comparison
- Day 30 CT dosimetry standard
- AAPM/ESTRO consensus on post-implant timing

## Key Findings
- TG-43 / NIST 1999 (or 2017 NIST update) is the standard formalism
- mPD is the formal prescription for LDR
- D90 and V100 are key quality metrics
- Modern LDR uses real-time TRUS planning
- Post-implant CT at 30 days standard
