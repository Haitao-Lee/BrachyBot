## Issues Found & Fixed

### 🔴 Critical Issues

#### 1. Tool Parameter Validation (filesystem_browser)
- **Issue:** LLM sent invalid `action: "&t"` to filesystem_browser
- **Fix:** Added validation: `if p.get("action") not in ("list", "info"): p["action"] = "list"`
- **Status:** ✅ Fixed

#### 2. Raw Tool Call in Response (Anthropic format)
- **Issue:** LLM returned `[{'type': 'tool_use', ...}]` as final response text
- **Fix:** Added Format 5 pattern for Anthropic tool_use blocks; cleaned text before yielding
- **Status:** ✅ Fixed

#### 3. Memory Artifact Leaking to User
- **Issue:** `[Called ui_inspector]` appeared in chat response
- **Fix:** Multi-layer fix: response cleaning, context summary cleaning, conversation history filtering
- **Status:** ✅ Fixed

#### 3b. TOOL_CALL Block Not Cleaned
- **Issue:** `[TOOL_CALL]...[/TOOL_CALL]` blocks appeared in response
- **Fix:** Changed pattern to `\[TOOL_CALL\].*?\[/TOOL_CALL\]` with `re.DOTALL`
- **Status:** ✅ Fixed

#### 3c. Streaming Text Chunk Leaking Tool Calls
- **Issue:** Text chunks yielded before full response cleaning, causing tool_use JSON to leak
- **Fix:** Accumulate full content, clean it, then yield only incremental new text while skipping tool_call patterns
- **Status:** ✅ Fixed

#### 3d. Python Set Format Tool Use Leak
- **Issue:** `{'tool_use', 'id': '...', 'name': '...', 'params': {...}}` format not cleaned
- **Fix:** Added regex pattern for Python set/dict format tool_use
- **Status:** ✅ Fixed

### 🟡 Medium Issues

#### 4. Welcome Message in English
- **Fix:** Changed to English: "👋 Welcome to BrachyBot!..."
- **Status:** ✅ Fixed

#### 5. Execution Trace Exposed Internal Details
- **Fix:** Added friendly English labels map
- **Status:** ✅ Fixed

#### 6. System Prompt Too Verbose / Weak Stopping Rules
- **Fix:** Streamlined system prompt, added explicit safety rules and stopping rules
- **Status:** ✅ Fixed

#### 7. Internationalization
- **Issue:** System prompts and UI in Chinese
- **Fix:** Translated to English, added language matching instruction
- **Status:** ✅ Fixed

#### 8. Benchmark Quality Issues
- **Issue:** 246 duplicate test cases (24.6% waste)
- **Fix:** Deduplicated all benchmark files, removed copy-paste bugs
- **Status:** ✅ Fixed

## Architecture Notes

- **LLM Provider:** MiniMax (Anthropic-compatible API)
- **Tool System:** 16 registered tools with OpenAI function calling format
- **Streaming:** SSE with real-time text chunks and step events
- **Memory:** Layered L0-L4 with experience recording
- **Self-Evolution:** Reflexion pattern with skill crystallization
- **Internationalization:** English system prompts, language matching for user responses
